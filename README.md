## Installation and setup
- Download and install PostgreSQL from the official website: https://www.postgresql.org/download/
- Follow the installation instructions for your operating system.
- After installation, you can access the PostgreSQL command line interface (psql) or use a graphical tool like pgAdmin to manage your databases.
- We can use terminal to create a new database and user:
  ```bash
  psql -U postgres
  CREATE DATABASE mydatabase;
  ```
